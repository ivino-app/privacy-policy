1. Coleta de Dados

O aplicativo EnoLink coleta as seguintes informações para funcionamento adequado:

  • Dados de cadastro (nome, e-mail)
  • Dados de uso (como você interage com o app)
  • Não armazenamos informações sensíveis sem seu consentimento

2. Uso dos Dados

Os dados são utilizados para:

  • Melhorar a experiência do usuário
  • Personalizar conteúdo
  • Garantir segurança e cumprir obrigações legais
  • Gerenciar assinaturas do serviço

3. Compartilhamento com Terceiros

Não vendemos ou compartilhamos seus dados com terceiros, exceto:

  • Provedores de serviços essenciais (ex: processamento de pagamentos)
  • Quando exigido por lei

4. Direitos do Usuário

Você pode:

  • Solicitar acesso, correção ou exclusão dos seus dados
  • Revogar consentimentos a qualquer momento
  • Opinar sobre assinaturas através das configurações da sua conta
